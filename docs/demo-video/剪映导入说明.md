# DuMe Demo — 剪映导入说明

本目录提供 **3 种可直接使用的剧本文件**（UTF-8 编码），总时长约 **3 分钟**。

| 文件 | 用途 | 推荐场景 |
|---|---|---|
| `DuMe_Demo.srt` | **导入字幕**（带时间轴） | 剪映专业版 / CapCut 桌面 / Web |
| `DuMe_Demo_图文成片.txt` | **图文成片旁白** | 剪映 App / 专业版「图文成片」 |
| `DuMe_Demo_分镜导入.txt` | **分镜脚本**（画面+旁白+字幕） | 手动剪辑、或复制到 AI 文案框 |

---

## 方式一：导入 SRT 字幕（推荐，有时间轴）

**适用：** 剪映专业版、CapCut 桌面版、CapCut Web  
**文件：** `DuMe_Demo.srt`

### 剪映专业版

1. 新建项目，可先拖入占位视频或录屏素材（总长约 3 分钟）。
2. 顶部菜单 **文本** → **导入字幕**（或 **本地字幕** → **导入字幕**）。
3. 选择 `DuMe_Demo.srt`。
4. 导入后可用 **文本朗读** 生成 AI 配音，并与字幕对齐。
5. 按分镜替换/叠加画面素材（见 `DuMe_Demo_分镜导入.txt` 中的「画面」列）。

### 注意

- 文件必须为 **UTF-8** 编码（本仓库已按此保存）。
- 若乱码：用记事本 / VS Code 另存为 UTF-8 后重新导入。
- **手机版剪映** 可能不支持直接导入 SRT，请用电脑版或下方「图文成片」方式。

---

## 方式二：图文成片（一键 AI 成片）

**适用：** 剪映 App、剪映专业版  
**文件：** `DuMe_Demo_图文成片.txt`

1. 打开剪映 → **图文成片**（或首页 AI 创作里的图文成片）。
2. 选择 **自由编辑文案** / **自定义文案**。
3. 打开 `DuMe_Demo_图文成片.txt`，**全选复制**，粘贴到文案框。
4. 选择 **智能匹配素材** 或 **使用本地素材**（建议后者并替换为产品录屏）。
5. 生成后进入时间线，用 `DuMe_Demo_分镜导入.txt` 对照补画面。

> 图文成片**不支持**从文件按钮上传 txt，需复制粘贴全文。

---

## 方式三：分镜脚本（画面 + 旁白 + 字幕）

**适用：** 自己剪辑、或给剪映 AI 写「画面描述」  
**文件：** `DuMe_Demo_分镜导入.txt`

每条分镜含三行：

- **画面**：给素材搜索 / AI 配图 / 屏录参考  
- **旁白**：配音稿  
- **字幕**：屏幕短字幕  

### 剪映专业版「导入文本」

1. **文本** → **导入文本** → 选择 `DuMe_Demo_分镜导入.txt`（若版本支持）。
2. 或打开文件，**只复制「旁白」行** 到文本轨道，**「字幕」行** 单独做标题。

---

## 建议工作流（效果最好）

1. 用 **图文成片** 或 **文本朗读** 生成旁白音轨（`DuMe_Demo_图文成片.txt`）。
2. 用 **SRT** 导入字幕并微调时间（`DuMe_Demo.srt`）。
3. 分镜 4–6 用 **dume-liart.vercel.app** 真实录屏替换 AI 素材。
4. 结尾字卡：`dume-liart.vercel.app`（VPN 提示仅字幕小字，可不配音）。
