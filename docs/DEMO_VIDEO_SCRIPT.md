## 分镜 4｜智能排期（0:50–1:25）

**画面：** 用户输入「明天下午写论文 2 小时，晚上复习高数 1 小时」→ AI 解析任务卡片（DDL、时长、优先级）→ 日历出现 25 分钟专注块 + 5 分钟休息。

**旁白：**  
不用一条条填表。用自然语言描述任务，AI 会解析截止时间、时长和优先级，再按你的可用时间，生成番茄钟式日程——25 分钟专注，5 分钟休息。排错了？日历上直接改。

**字幕：** 自然语言 → 结构化任务 → 智能排期

**屏录建议：** 登录 → Schedule → 输入任务 → AI 排期 → 展示日历块。

## 分镜 5｜沉浸监督（1:25–2:10）★ 重点

**画面：** 选任务进入监督 → 监督官卡片（尤里教官 / 顾姐 / 林风）→ 摄像头人脸登记 → 全屏倒计时 25:00 → 检测到分心触发警告视频 → 专注币 +1。

**旁白：**  
选好任务，挑一位监督官——比如铁血风格的尤里教官。开启摄像头完成人脸登记后，进入专注块。分心、离位、玩手机？本地视觉会在浏览器里实时判断——画面不上传云端，隐私优先。监督官会用预制视频立刻给你反馈；累计警告可能导致本轮专注失败。完成一块，还能获得专注币。

**字幕：** 本地 CV · 不上传画面 · 角色即时反馈

**屏录建议：** 选尤里 → 监督模式 → 倒计时与 strike →（可选）拿手机触发 warn → 完成专注块。

**旁白备选（无法演示摸鱼时）：**  
系统会根据分心程度分级提醒，累计警告可能导致本轮专注失败。

## 剪映使用说明

1. **图文成片**：按分镜 1–8 逐段粘贴「画面」描述；旁白用上方全文或分段生成配音。
2. **更可信**：分镜 4–6 建议用真实屏录替换 AI 画面（画中画叠在旁白段上）。
3. **超时**：若超过 3 分钟，优先压缩分镜 2、分镜 7 各 5–8 秒。
4. **产品要点（勿夸大）**：LLM 排期与解析；浏览器本地 CV；监督官预制视频反馈；Injective 为规划中。
